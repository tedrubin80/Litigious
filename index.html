Legal Estate Management

Setup Required: The application needs to be built before deployment.

Deployment Instructions:

  1. Build the frontend application on your local machine:

    cd frontend
npm install
npm run build

  2. Upload the contents of frontend/build/ directory to your server's root directory

  3. The app should then be accessible at your domain root

Note: Make sure to upload the contents of the build folder (index.html, static/, etc.), not the build folder itself.
