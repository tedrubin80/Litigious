Litigious is a modular LMS for law firms and legal tech builders — case management, time tracking, documents, client portal, AI-assisted workflows, and migration from Clio, MyCase, and PracticePanther.

What you get

Case & client management

Full lifecycle from intake through close, roles, deadlines, and client portal access.

Billing & time

Billable hours, invoicing, and integrations with major payment providers.

Documents & AI

Secure uploads, previews, and optional multi-provider AI document generation.

LMS migration

Import adapters for Clio, MyCase, PracticePanther, and CSV exports with dry-run preview.

Self-host or cloud

Docker locally, Railway for full stack, Vercel for frontend — documented and templated.

White-label ready

Deploy under your brand, domain, and module selection for resellers and consultants.

Deploy in minutes

Config-as-code is included. Fork, set env vars, deploy.

  1. Fork tedrubin80/Litigious
  2. npm run setup — install deps and copy env templates
  3. Railway: Postgres + backend/ + frontend/ (see deploy guides)
  4. Optional: npm run seed:demo for showcase data

Quick start (developers)

git clone https://github.com/tedrubin80/Litigious.git
cd Litigious
npm run setup
# edit backend/.env
npm run db:push && npm run dev
